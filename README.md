Nodejs-Boilerplate
====================

## Setup
### Requirements

Node `">=5.0.0"` (use brew or install it from [here](http://nodejs.org/download/))

```bash
brew install node
```

### Clone the repository

```bash
git clone https://github.com/andyfrontend/nodejs-boilerplate.git
```

### Dependencies

```bash
npm install
```
with yarn

```bash
yarn install
```

## Develop

Create a bundle

```bash
npm run dev:build
```

Run server

```bash
npm run serve
```
